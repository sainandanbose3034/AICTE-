About This Project

Understanding the dataset and methodology

📊

The Dataset

The Housing Prices Dataset from Kaggle contains 545 records with 13 features covering property area, number of rooms, amenities, and location preferences. Licensed under CC0 Public Domain.

🧠

Multiple Linear Regression

Our model uses Ordinary Least Squares (OLS) to find the best-fit hyperplane through the feature space. The formula: ŷ = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ

⚡

Client-Side Prediction

The model coefficients are exported from Python to JSON. All predictions happen directly in your browser using JavaScript — no server requests needed for inference.

🎯

Feature Engineering

Binary features (yes/no) are encoded as 1/0. Categorical features (furnishing status) are one-hot encoded. All 12 original features are used for prediction.
